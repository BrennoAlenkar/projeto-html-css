HTML... é hypertext markup language, o html é a estrutura das paginas web, marcando com tags o que é cada coisa, é a linguagem mais básica da web.

HTML significa linguagem de Marcação de hiper texto, os Hiper textos se refere aos links, exemplo a baixo de um link.
